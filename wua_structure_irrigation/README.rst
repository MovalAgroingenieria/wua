=========================================
Scheleton for base_wua_irrigation modules
=========================================

This module is the common scheleton for the base_wua_pressurized_irrigation
and base_wua_gravity_irrigation modules.

Description
===========

The module will be installed automatically when installing...
 
* base_wua_pressurized irrigation, or
* base_wua_gravity irrigation
