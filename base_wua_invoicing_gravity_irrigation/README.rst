===================================================================
Water Users Association Invoicing (gravity irrigation consumptions)
===================================================================

This module allows you to invoice gravity irrigation, making a set of
invoices based on a selection of consumptions of a particular type of
water.

Description
===========

Global Invoicing based on:

* Gravity Irrigation Consumptions of a type of water.
