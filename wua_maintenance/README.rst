============================================================
Maintenance for Water Users Associations
============================================================

Introduction of team tags and categories for Water Users Associations,
so that grouping and maintenance integration with WUA infrastructure is
much easier.

Description
===========

This module will allow you to create maintenance teams, tags for Water Users Associations
teams and categorize them based on its relationship with Water Users Associations
infrastructure.

Functionality:

* Easy creation of maintenance teams linked to WUA infrastructure.
* Use of tags for maintenace teams.
* Implementation of categories for the classification of WUA infrastructure.
