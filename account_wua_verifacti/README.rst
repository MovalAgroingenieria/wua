=====================
Account WUA Verifactu
=====================

This module provides Verifactu QR code integration for Water Users Association invoice reports.

Description
===========

This module adds the Verifactu QR code to the invoice reports of Water Users Associations (WUA).
The QR code facilitates quick access to invoice details and verification through the Verifactu platform.
