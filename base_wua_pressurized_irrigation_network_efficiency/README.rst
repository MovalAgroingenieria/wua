=========================================
Pressurized Irrigation Network Efficiency
=========================================

WUA module for the evaluation of the efficiency of the pressure irrigation
network.

Description
===========

This module introduces the measurement of the efficiency of water distribution.
It is about comparing:

* the volume of water entering a branch, with

* the sum of volumes of water that come out through the intakes that are connected with that branch.

As a result of this comparison, each branch consumption can be associated with
a ratio that quantitatively expresses this efficiency. This ratio will be
between the extreme values 0 (all the water is lost before reaching the
intakes) and 1 (there is no water loss).
