====================================================================================
Pressurized Irrigations Requests for Water Users Associations With Provisional Quota
====================================================================================

This module adds quota management for Pressurized Irrigation Requests.

Description
===========

Requests for Pressurized Irrigation. Now presresconsumptions will generate
related control readings which will detract from the quota with each reading.

If remotecontrol is installed, and exists some presresconsumption related, the
real value of the presresconsumption and the controlreading will be obtained
from the remote control device.
