===========================================================
Water Users Association Invoicing (fertilizer consumptions)
===========================================================

This module allows you to invoice fertilizers, making a set of
invoices based on a selection of consumptions of a particular type of
fertilizer.

Description
===========

Global Invoicing based on:

* Fertilizer Consumptions.
