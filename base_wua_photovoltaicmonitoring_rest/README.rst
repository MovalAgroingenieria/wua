Water Users Association Irrigation (REST API for photovoltaic monitoring)
=========================================================================

Base module for other modules that implement communication dialogues with
photovoltaic monitoring systems (with RESP API).

Description
===========

The module is not installable and is not an application. It is a base module
for other modules that implement communication processes with the real
photovoltaic monitoring systems. This communication is based on API REST.

The context is the pumping installations.

Functionality:

* Obtaining data of electrical generation measurements.
* Automation of readings.
* Access to native software of photovoltaic monitoring.
