========================================
Gravity Irrigation with Quota Management
========================================

New improvements to the user experience, such as get the current balances in
form view of watering requests.

Description
===========

After choosing the partner of a watering request, the program will calculate
the partner's current balance for each hydric superproduct, in the active
agricultural season.
