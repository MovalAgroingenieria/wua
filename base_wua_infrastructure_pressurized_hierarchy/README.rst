======================================================
Hierarchical Infrastructure For Pressurized Irrigation
======================================================

This module allows you to manage hierarchically the infrastructure of water users associations, for pressurized irrigation.

Description
===========

Hierarchical infrastructure management for:

* Hydraulic sectors.
* Water pipes.
