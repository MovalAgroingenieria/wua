==================================================================
Water Users Association Invoicing (irrigation report consumptions)
==================================================================

This module allows you to invoice irrigation reports, making a set of invoices
based on a selection of irrigation reports of the current agricultural season.

Description
===========

Global Invoicing based on water consumption by irrigation reports.
