===========================================================
Water Users Association Irrigation (interface with Regacom)
===========================================================

Interface of Moval Regadio with the Regacom SQL database
based on a REST API

Description
===========

Interface of Moval Regadio with the Regacom SQL database,
based on a REST API, and with the functional schema of the
base_wua_remotecontrol_rest module.

Regacom is a SQL database system for remote control and monitoring
of irrigation systems.
