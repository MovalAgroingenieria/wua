==============================================================================
Water Users Association Invoicing (fixed and variable surcharges for invoices)
==============================================================================

This module allows you to invoice fixed and variables surcharges for,
unpayed invoices.

Description
===========

Global Invoicing based on:

* Fixed and variable surcharges for unpayed (or payed) invoices.
