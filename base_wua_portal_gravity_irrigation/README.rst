=========================================
WUA: Gravity Irrigation Portal Experience
=========================================

Portal module that allows irrigators to consult their gravity irrigation
consumptions through the customer portal at ``/my/gravconsumptions``.

Features
--------

* List gravity irrigation consumptions grouped by watering period and state
* Filter by period, state, and free-text search (watering, parcel,
  irrigation ditch)
* Date formatting in DD/MM/YYYY HH:MM:SS
* Pagination support
* Compatible with ``base.group_portal`` and ``base_wua.group_wua_portal_user``
