============================
Management of updated quotas
============================

Quota management with updated balances for pressurized irrigation (in real time, or similar).

Description
===========

A new type of balances will be updated for each import of control readings, without changing conventional balances. The purpose is to get these balances before (or long before) carryng out the import of normal readings.
