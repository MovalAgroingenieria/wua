================================================================
Water Users Association: Grafana integration for the NDVI graphs
================================================================

Integration with Grafana of the Moval-Regadío Sentinel-Hub API Interface.

Description
===========

This module replaces Bokeh graphics with graphics generated in Grafana with the
same data. This allows a better visualization of the data and an interaction
with the generated graph (zoom to the time frame, annotations, etc.)
