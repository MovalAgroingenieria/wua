=====================================================================
Water Users Association Irrigation (interface with BATCHLINE IRRIWEB)
=====================================================================

Interface of Moval Regadio with the control remote BATCHLINE IRRIWEB
based on a REST API

Description
===========

Interface of Moval Regadio with the control remote BATCHLINE IRRIWEB,
based on a REST API, and with the functional schema of the
base_wua_remotecontrol_rest module.

Installation
============

If the integration of frames for readings, consumptions and scheduing
is enabled, then it is mandatory to have the following PHP on a
web server: framebatchlineirriweb.php.

 Example::

    http://127.0.0.1/framebatchlineirriweb.php
