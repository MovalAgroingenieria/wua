=======================================================================
Certificate Management for Water Users Association (multiple signature)
=======================================================================

Certificate management for any water users association, with simple or
multiple signature

Description
===========

This module allows to get a certificate of belonging to a water users
association. A multiple signature is possible, and show the signature in
the PDF file if you set the "show signature" parameter.

IMPORTANT: To display a background image over the signature, place the image
in the certificate folder.
