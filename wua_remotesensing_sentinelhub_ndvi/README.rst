=============================================================================
Water Users Association (capture of NDVI index for parcels from Sentinel-Hub)
=============================================================================

Interface of Moval-Regadío with the Sentinel-Hub API, to get the NDVI index
of parcels.

Description
===========

This module is based on the interface of base_wua_remotesensing_sentinelhub,
and it is directly installable. The module does the interface with the
API FIS of Sentinel-Hub, to get the minimum, mean and maximum values of the
NDVI index, for each parcel and each day.
