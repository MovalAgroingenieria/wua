===============================================
Water Users Association: Pump Infrastructure
===============================================

This module allows you to manage the pumping infrastructure of water users associations.

Description
===========

Management of the pumping infrastructure of a water user association:

* Photovoltaicplants.
* Pumpunits.
* Pumps Groups.
