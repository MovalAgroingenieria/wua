======================================================
Water Users Association Infrastructure (C.R.Almassora)
======================================================

Custom Moval-Regadío for C.R. Almassora.

Description
===========

Moval-Regadío customization for C.R.Almassora:

* Only a subparcel for each parcel.
* Mandatory irrigation ditch, and its code must be equal to parcel code.
* Only a water connection for each parcel.
* Payments to irrigation workers.
* Reports: Irrigation Request Labels, and Watering Consumptions.
