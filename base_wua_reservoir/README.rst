===================================================
Water Users Association: Water Reservoir Management
===================================================

Base module for the management of Water Reservoirs in Water Users Associations.

Description
===========

This module allows the integral management of Water Reservoirs.



Installation
============

You need to install the python bokeh library::

    pip install bokeh==0.12.7
