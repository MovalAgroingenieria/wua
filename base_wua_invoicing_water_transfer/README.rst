=========================================================================
Water Users Association Invoicing (irrigation volume for water transfers)
=========================================================================

This module allows you to invoice contracted irrigation volumes, making a set
of invoices based on a selection of water transfers of the current
agricultural season.

Description
===========

Global Invoicing based on:

* Contracted irrigation volume of a selection of water transfers.
