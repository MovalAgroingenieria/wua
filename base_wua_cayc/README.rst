======================================================
Water Users Association Management: CAYC Specification
======================================================

This module adds specific functionality for CAYC Water User Associations.

Description
===========

Moval Regadío customization for CAYC clients:

* WUA code on res_company.
* Special treatment of parcels with gis data.
* Etc.
