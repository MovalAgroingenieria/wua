======================================
Water Users Association Infrastructure
======================================

This module allows you to manage the infrastructure of water users associations.

Description
===========

Management of a water user association infrastructure.

* Irrigation ditches.
* Irrigation gates.
* Irrigation sheds.
* Water connections.
* Etc.
