==============================
WUA Performance Analyzer Rules
==============================

Description
===========

This module provides a set of rules to analyze the performance of WUA (Water User Associations) operations.
It is designed to work with the Smile Performance Analyzer module, allowing users to monitor and
optimize their processes effectively.
