============================================================
Management of notification sets for Water Users Associations
============================================================

Management of the the entire process of implementing a notification set.

Description
===========

This module will allow you to design and create notification sets and send
these notifications in bulk to all or part of the partners.

Functionality:

* Template design for a notification type.
* Implementation of a notification set, mapped to multiple partners.
* Validation and sending of notifications to partners.
