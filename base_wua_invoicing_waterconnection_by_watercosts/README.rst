=========================================================================================
Water Users Association Invoicing (water connections, based on percentage of water costs)
=========================================================================================

This module allows you to invoice a set of water connections from the
water-costs percentage of partners. 

Description
===========

Invoicing similar to category 5 (water connections), but the percentage
to apply for each partner is not the percentaje of other costs, else the
percentage of water costs.
