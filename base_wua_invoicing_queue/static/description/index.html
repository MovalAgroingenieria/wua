WUA Invoicing – Calculate in background (queue)

This module adds a Calcular en paralelo (Calculate in parallel) button to invoice sets. It runs the same calculation as Calcular but in a queue_job worker so the interface does not block. It depends on backend_process_status so a green blinking indicator in the top bar shows when the calculation is running or enqueued.

Usage

  • Open an invoice set in draft, configured and not yet generated.
  • Click Calcular en paralelo and confirm.
  • The calculation is enqueued; follow the status via the indicator in the top bar.
  • Ensure a queue job worker is running for your database.
