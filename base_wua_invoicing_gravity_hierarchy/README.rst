======================================================
Water Users Association Management: Invoicing Gravity Hierarchy
======================================================

This module adds specific functionality for Invoicing Gravity Hierarchy Water User Associations.

Description
===========

Invoicing customization to add irrigationditches and drainageditches of different levels to wua.invoiceset.line.parcel

* Adds irrigationditch_01_id, irrigationditch_02_id
* Adds drainageditch_id, draingeditch_01_id, draingeditch_02_id
