=================================
Water Users Association Invoicing
=================================

This module allows you to invoice WUA products globally, making a set of
invoices based on a selection of partners, a selection of packages,
a selection of water connections, etc.

Description
===========

Global Invoicing based on:

* Parcels Selection.
* Partners Selection.
* Water Connections Selection.
* Irrigation Gates Selection.
